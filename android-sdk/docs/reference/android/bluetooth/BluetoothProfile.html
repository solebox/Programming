BluetoothProfile

android.bluetooth.BluetoothProfile
Known Indirect Subclasses
BluetoothA2dp, BluetoothHeadset, BluetoothHealth
BluetoothA2dp This class provides the public APIs to control the Bluetooth A2DP profile. 
BluetoothHeadset Public API for controlling the Bluetooth Headset Service. 
BluetoothHealth Public API for Bluetooth Health Profile. 

Class Overview

Public APIs for the Bluetooth Profiles.

Clients should call getProfileProxy(Context, BluetoothProfile.ServiceListener, int), to get the Profile Proxy. Each public profile implements this interface.
