Handler

extends Object
java.lang.Object
   ↳ java.util.logging.Handler
Known Direct Subclasses
MemoryHandler, StreamHandler
MemoryHandler A Handler put the description of log events into a cycled memory buffer. 
StreamHandler A StreamHandler object writes log messages to an output stream, that is, objects of the class OutputStream. 
Known Indirect Subclasses
ConsoleHandler, FileHandler, SocketHandler
ConsoleHandler A handler that writes log messages to the standard output stream System.err. 
FileHandler A FileHandler writes logging records into a specified file or a rotating set of files. 
SocketHandler A handler that writes log messages to a socket connection. 

Class Overview

A Handler object accepts a logging request and exports the desired messages to a target, for example, a file, the console, etc. It can be disabled by setting its logging level to Level.OFF.

protected void reportError (String msg, Exception ex, int code)

Since: API Level 1

Reports an error to the error manager associated with this handler, ErrorManager is used for that purpose. No security checks are done, therefore this is compatible with environments where the caller is non-privileged.

Parameters
msg the error message, may be null.
